AI-Integrated Rehabilitation System

HardwareAI SystemsEmbedded CMobile

Overview

This collaborative project developed a wearable smart glove system for clinical neuro-rehabilitation, targeting patients recovering from stroke or neurological conditions affecting fine motor control. The system integrates embedded sensor hardware with an AI-driven mobile application to deliver gamified, real-time therapy exercises, increasing patient motivation and enabling clinicians to monitor progress remotely.

System Architecture

  • Smart Glove Hardware — Flex sensors, IMU (accelerometer + gyroscope), and haptic feedback actuators embedded in a wearable glove form factor, communicating over Bluetooth Low Energy (BLE).
  • Embedded Firmware — Written in C for a microcontroller platform, responsible for real-time sensor fusion, data buffering, and BLE transmission.
  • AI Feedback Engine — A gesture classification model maps sensor streams to discrete hand gestures, scoring therapy exercise execution and adapting difficulty dynamically.
  • Mobile Gaming Interface — A mobile application renders interactive rehabilitation exercises as games, receiving gesture classification results to drive in-game actions and provide immediate visual/haptic feedback.

Key Contributions

  • Designed and implemented the embedded firmware for real-time sensor data acquisition and BLE communication.
  • Integrated the AI gesture classification pipeline between the hardware data stream and the mobile application layer.
  • System demonstrated measurable improvement in patient engagement metrics compared to traditional passive rehabilitation methods in preliminary clinical observations.
