Digital Watermarking Protocol

PythonCryptographyDWT-SVDSignal Processing

Overview

This project presents a hybrid digital watermarking protocol that combines the robustness of frequency-domain transforms with the unpredictability of chaotic encryption to achieve imperceptible yet highly resilient media authentication. The system targets a core challenge in digital forensics: embedding an ownership signature into multimedia content without degrading its perceptual quality.

Methodology

The approach integrates two well-established signal processing techniques:

  • Discrete Wavelet Transform (DWT) — decomposes the host image into frequency sub-bands, allowing the watermark to be embedded in perceptually significant regions.
  • Singular Value Decomposition (SVD) — applied to the selected sub-band coefficients, modifying singular values to encode the watermark in a mathematically stable manner.
  • Chaotic Encryption (Logistic Map) — the watermark itself is scrambled using a logistic map-based pseudo-random key generator before embedding, dramatically increasing resilience against extraction attempts.

Key Contributions

  • Demonstrated high imperceptibility (PSNR > 40 dB) while maintaining robust extractability after JPEG compression, cropping, and noise attacks.
  • Chaotic encryption layer significantly increases security against blind watermark extraction.
  • Implemented a complete Python-based pipeline for embedding and blind extraction without access to the original host image.
