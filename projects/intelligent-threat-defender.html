Intelligent Threat Defender

JavaScriptMachine LearningNLPBrowser Extension

Overview

Intelligent Threat Defender is a browser extension designed to classify and block malicious URLs in real-time at the client side. Rather than relying solely on blocklists — which cannot keep up with the rate of new phishing and malware domains — the system employs a machine learning pipeline to detect novel, previously unseen threats based on URL structure and lexical properties.

Methodology

  • Feature Extraction — URLs are parsed into lexical features (length, digit ratio, special character count, TLD entropy) and host-based features (domain age, IP presence, redirect depth).
  • ML Classification — A lightweight gradient-boosted decision tree model (optimized for inference speed) classifies each URL as benign or malicious.
  • Real-time Interception — The browser extension intercepts navigation events via the WebExtensions API before the page loads, querying the local model and blocking flagged URLs with a warning page.

Key Contributions

  • Achieved > 95% classification accuracy on benchmark phishing datasets with sub-5ms inference time per URL.
  • Entirely client-side inference — no data is sent to external servers, preserving user privacy.
  • Lightweight model footprint (< 500 KB), suitable for browser extension constraints.
  • Extensible feature pipeline supporting future addition of NLP-based URL tokenisation.
